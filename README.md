# 🐷 小猪的记账本

一个轻量、好看、可离线使用的个人记账 PWA。

## 功能

- 💱 15 种主流货币（港币 / 人民币 / 美元 / 欧元 / 日元 等）
- 📅 日历视图，按消费强度显示颜色（轻度 / 适中 / 偏高 / 超支）
- ⚠️ 每日限额提醒
- 🧮 内置计算器
- 📊 月度对比、消费洞察、分类饼图
- 🔍 账单搜索
- 🔁 一键复制账单
- 💾 数据备份与恢复（JSON 文件）
- 📤 导出 Excel 账单
- 📱 PWA，可添加到手机桌面离线使用

## 使用

直接打开 [index.html](./index.html) 即可在浏览器中使用。

或访问已部署的版本：通过 GitHub Pages 自动部署。

## 技术栈

- 纯 HTML + CSS + JavaScript（单文件）
- Chart.js 用于图表
- SheetJS 用于 Excel 导出
- Service Worker 实现离线缓存
- LocalStorage 存储数据
